# 🧮 Calculator App

A simple and responsive Calculator application built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations with a clean and user-friendly interface, providing fast and accurate calculations.

## 📸 Screenshot

![Calculator App](image(4).png)

## 🚀 Features

- Perform basic arithmetic operations
  - ➕ Addition
  - ➖ Subtraction
  - ✖ Multiplication
  - ➗ Division
  - % Percentage
- Clear all input
- Delete the last entered character
- Responsive and attractive user interface
- Real-time calculations
- Lightweight and easy to use

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/calculator-app.git
```

2. Navigate to the project folder:

```bash
cd calculator-app
```

3. Open **index.html** in your preferred web browser.

No additional installation or dependencies are required.

## 💡 How It Works

1. Click the number buttons to enter values.
2. Select an arithmetic operator (+, -, ×, ÷, %).
3. Press **=** to display the result.
4. Use **C** to clear all input.
5. Use **←** to delete the last entered character.
